# InView

## Installation

1. Include dependencies *jQuery* & *Waypoints*
    ```code
    <script src="jquery.min.js"></script>
    ```
    ```code
    <script src="jquery.waypoints.min.js"></script>
    ```
2. Include *InView*
    ```code
    <script src="jquery.inview.js"></script>
    ```
3. Initialize the plugin
    ```code
    $('.wrapper').inView();
    ```

## Options

*Offset:*
```code
$('.wrapper').inView({
    offset: 60
});
```
